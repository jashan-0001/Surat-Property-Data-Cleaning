# Surat Property Data Cleaning Project

## Project Overview

This project demonstrates data cleaning techniques using Python and Pandas on a Surat property dataset.

## Dataset

- Original Dataset: `surat_uncleaned.csv`
- Cleaned Dataset: `surat_cleaned.csv`

## Data Cleaning Steps

- Removed duplicate records
- Handled missing values
- Filled categorical missing values with appropriate defaults
- Filled numerical missing values using the median
- Removed extra spaces from text columns
- Standardized text formatting
- Cleaned the `price_per_sqft` column
- Reset the index
- Saved the cleaned dataset

## Technologies Used

- Python
- Pandas
- Visual Studio Code
- GitHub

## Files

- `clean_surat_data.py` – Python script used for cleaning
- `surat_uncleaned.csv` – Original dataset
- `surat_cleaned.csv` – Final cleaned dataset
